# โครงสร้างคณะและหลักสูตร มหาวิทยาลัยสวนดุสิต (Faculty Structure Analysis)

**URL**: [https://www.dusit.ac.th/home/faculties-and-departments](https://www.dusit.ac.th/home/faculties-and-departments)
**Date**: 2026-02-13

## 1. ภาพรวม (Overview)
มหาวิทยาลัยสวนดุสิตประกอบด้วยคณะและโรงเรียนวิชาชีพหลัก 9 หน่วยงาน ซึ่งแต่ละหน่วยงานมีหลักสูตรย่อยที่ AI Agent จำเป็นต้องแยกแยะบริบทให้ถูกต้อง

## 2. โครงสร้างหน่วยงาน (Faculty Breakdown)

| หน่วยงาน (Faculty/School) | หลักสูตรเด่น/สาขาวิชา (Key Programs) | หมายเหตุสำหรับ AI |
| :--- | :--- | :--- |
| **1. คณะครุศาสตร์** (Education) | ปฐมวัย, ประถมศึกษา, วิชาชีพครู, ล่ามภาษามือ | ข้อมูลมักเป็นกฎเกณฑ์ใบประกอบวิชาชีพครู |
| **2. คณะวิทยาศาสตร์และเทคโนโลยี** | วิทยฯคอม, เครื่องสำอาง, อาชีวอนามัย, สิ่งแวดล้อม, วิทยฯการแพทย์, Cyber Security | มีศัพท์เทคนิคและ Lab Safety Rules |
| **3. คณะวิทยาการจัดการ** | การจัดการ, นิเทศศาสตร์, เลขานุการแพทย์, การบัญชี, ธุรกิจบันเทิง, ธุรกิจจีน-อาเซียน | เน้นเรื่องฝึกงานและธุรกิจ |
| **4. คณะมนุษยศาสตร์และสังคมศาสตร์** | จิตวิทยา, ภาษาไทย, อังกฤษ, จีน, เกาหลี | ข้อมูลเกี่ยวกับภาษาและการสื่อสาร |
| **5. คณะพยาบาลศาสตร์** | พยาบาลศาสตร์, ผู้ช่วยพยาบาล | *ข้อมูลความปลอดภัยสูง* (Patient/Trainning data) |
| **6. โรงเรียนการท่องเที่ยวและการบริการ** | ท่องเที่ยว, โรงแรม, ธุรกิจการบิน, ออกแบบนิทรรศการ | เน้นบุคลิกภาพและตารางการบิน/ฝึกงาน |
| **7. โรงเรียนการเรือน** (Culinary Arts) | เทคโนโลยีอาหาร, คหกรรม, โภชนาการ | **จุดแข็ง SDU** มีสูตรอาหารและมาตรฐานครัว |
| **8. โรงเรียนกฎหมายและการเมือง** | นิติศาสตร์, รัฐศาสตร์, รัฐประศาสนศาสตร์ | ข้อมูลกฎหมายและ พรบ. |
| **9. บัณฑิตวิทยาลัย** | หลักสูตรระดับปริญญาโท-เอก | เงื่อนไขการจบจะซับซ้อนกว่า ป.ตรี |

## 3. ข้อสังเกตสำหรับ R&D Data Pipeline
1.  **Sub-domains**: เกือบทุกคณะมี Website ของตัวเอง (Sub-domain) เช่น `nurse.dusit.ac.th`, `food.dusit.ac.th` ทำให้ Crawler ต้องตั้งค่า Scope ให้ครอบคลุม
2.  **External Platforms**: บางคณะใช้ Facebook Page เป็นช่องทางสื่อสารหลัก (เช่น นิเทศศาสตร์นวัตกรรม) ซึ่ง AI จะดึงข้อมูลยากกว่า Web ปกติ
3.  **Specific Formats**:
    - **โรงเรียนการเรือน**: อาจมีไฟล์ภาพเมนูอาหาร หรือวิดีโอสาธิต (Multimedia Content)
    - **คณะพยาบาล**: อาจมีตารางเวรฝึกปฏิบัติงาน (Excel/PDF Rotation Schedule)
